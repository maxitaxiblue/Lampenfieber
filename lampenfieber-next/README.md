# Lampenfieber

Dein Content hat Auftritt. KI-gestütztes Content-Tool für kleine und mittlere Unternehmen.

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" → Import this repository
4. Click "Deploy"

That's it. Vercel detects Next.js automatically.
